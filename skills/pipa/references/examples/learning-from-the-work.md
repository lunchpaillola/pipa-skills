# Learn From The Work Examples

## Lessons Capture

- `Lesson`: kickoff needed a named decision owner before delivery started
- `Evidence`: two blocked scope decisions in delivery notes
- `Next-time change`: add decision owner to define-work acceptance checks
- `Owner`: TBD
- `Reuse location`: project playbook or `TBD`
- `Follow-ups`: confirm handover/archive readiness

## Closure Request

Do not declare closure unless signoff, handover, and archive evidence are present or the user explicitly accepts the gap.
